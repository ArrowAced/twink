info

  • femboy-meter's prefix is "fem.", so for example "fem.update"
  • the official currency for femboy-meter is fembucks
  • your meter points cannot exceed 100 or go below 0
  • this is still a work in progress, please respect the time it takes to get out an update.

get started

to get into the database for femboy-meter (and have all of the other commands work), run "fem.meter". this command will show you your current femboy percentage as well as put you in the database.

meter

meter

update your femboy meter. ex: "fem.meter" would either send:

  1. a random number (if your username isn't in the database)
  2. a tweaked (increments or decrements by a random number through -10 and 10) version of your femboy meter

find

look at your own or someone else's femboy meter (without changing it). ex: "fem.find <username>" would either send:

  1. someone else's femboy meter
  2. your own femboy meter without changes or tweaks (if the first argument is your username)

currency

trade

[1 minute cooldown] trade your points from the femboy meter to fembucks based on the current stocks. ex: "fem.trade" would either send:

  1. conformation message that you have indeed traded your meter points for currency
  2. error message (not enough meter points to trade)
  3. error message (1 min cooldown)

buy

buy an item off of shop using fembucks.. ex: "fem.buy <item>" would either send:

  1. conformation message that you have bought the item
  2. error message (not enough fembucks)
  3. error message (non-existant item)

equip

equip an item you have. ex: "fem.equip <item>" would either send:

  1. conformation message that you have equipped the item (randomized)
  2. error message (you do not have the item)
  3. error message (you already have something like the item equipped)

this command will also raise your femboy meter points or decrease them.

unequip

unequip an item you have equipped. ex: "fem.unequip <item>" would either send:

  1. conformation message that you have unequipped the item (randomized)
  2. error message (you do not have an item like that equipped)

this command will also decrease your femboy meter by half of what it was raised/decreased.

readonly commands

shop

will return the current shop items you can buy. ex: "fem.shop"

update

get current update + notes on update. ex: "fem.update"

info

get info on runtime. ex: "fem.info"

help

send a link to this website. ex: "fem.help"

fembucks

view your current amount of fembucks. ex: "fem.fembucks"

items

look at all of your items (that you don't have equipped). ex: "fem.items" would either send:

  1. all of your items
  2. error message (you have no items)

equipped

get all of your items you have equipped/if you have an item equipped. ex: "fem.equipped <item: optional>"

stocks

see current stocks for selling points and fembucks back. ex: "fem.stocks"

notify

set if notifications for lucky stock are true or false. ex: "fem.notify"

events

  • database auto-saves every 5 seconds
  • the current stocks are updated with a random value every 30 seconds
    • sell: 0 - 20
    • return: 5 - 25
  • random% interest to fembucks is added to every user in database every 60 minutes/1 hour
    • random: 1 - 10
